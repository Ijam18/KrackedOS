# MajiOS

`MajiOS/` is the canonical memory-and-method architecture for `KRACKED_OS`.

It unifies:
- `core` for MAJI identity, relationship, session, and save behavior
- `method` for BMAD workflows and planning doctrine
- `engine` for memory persistence and retrieval
- `assets` for media as memory artifacts
- `skills` for reusable capability packs
- `system` for operating doctrine and activation guidance
- `index` for entrypoints, maps, and precedence rules
