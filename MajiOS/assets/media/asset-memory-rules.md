# Asset Memory Rules

## Media Classification
- brand memory
- UI and visual reference memory
- wallpaper and personalization memory
- external-source asset memory

## Operating Rules
- use media to recover project taste, direction, and visual decisions
- do not treat media as core code truth
- attach provenance when assets come from external packs or third parties
- keep media discoverable through indexes rather than scattered assumptions

## Precedence
If media conflicts with code or active product decisions, code and current decisions win.
