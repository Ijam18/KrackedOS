# 🧪 BMAD Distill Protocol
*Compress large Kracked_OS context into reusable briefings*

## When to Use
Trigger when user says:
- `bmad distill`
- "distill this"
- "make this easier to reload later"

## Goal
Produce a compact but high-signal brief from large design, planning, or repo documents.

## Process
1. Identify the source material
2. Extract:
   - facts
   - decisions
   - constraints
   - open questions
   - next actions
3. Remove repetition and low-signal prose
4. Format for future reuse
