# 📋 BMAD Plan Protocol
*Turn Kracked_OS goals into actionable work plans*

## When to Use
Trigger when user says:
- `bmad plan`
- "make a plan"
- "break this down"

## Goal
Convert a goal into a practical implementation sequence for Kracked_OS.

## Process
1. Identify the target outcome
2. Define success criteria
3. Break the work into phases
4. Sequence tasks by dependency and risk
5. Call out security, UX, or runtime constraints when important
